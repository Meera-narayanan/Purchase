ProductPurchase

A single page e-commerce application built with React. This project give  CRUD functionality Create, Read, Update, Delete the  products using data fetched from the Fake Store API. Users can view, edit, and delete products.


Navigation

cd fakestore


installation of depoendencies

npm install

start the server

cd fakestore
npm run

Open the app 

http://localhost:3000


Key design or technical decisions


# React Framework Chosen for component based architecture and efficient UI updates.

# React-router-dom is used to navigate between Home, Product Delete, and Edit pages.

# useState and useEffect manage product data .

# Usage of Fake Store API provides product data.

# CRUD Operations Add, Edit, and Delete update the UI  without page reloads.
